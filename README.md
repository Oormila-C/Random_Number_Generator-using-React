# Random_Number_Generator-using-React

## Description

The **Random Number Generator** is a simple React application built using the **useState** hook. It demonstrates React state management, event handling, and conditional rendering by generating a random number between **1 and 100** whenever the user clicks a button.

Initially, the application displays the message **"No number generated yet"**. After clicking the **Generate Random Number** button, a new random number is generated and displayed instantly. Every subsequent click generates a different random number, showcasing dynamic UI updates using React state.

## Features

* Built with React Functional Components
* Uses the **useState** hook for state management
* Generates a random number between **1 and 100**
* Updates the UI dynamically without refreshing the page
* Uses conditional rendering to display a placeholder message before the first number is generated
* Clean and responsive user interface with separate CSS styling
